# Question 4

**Compare `for`, `while`, and `do-while` loops with examples**

## For Loop
```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

## While Loop
```java
int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}
```

## Do-While Loop
```java
int i = 1;
do {
    System.out.println(i);
    i++;
} 
while (i <= 5);
```
